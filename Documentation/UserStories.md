# User Stories

## Doctor Stories

**US-01**  
As a doctor, I want to create digital prescriptions so that I can reduce manual errors.

**US-02**  
As a doctor, I want to access patient history so that I can prescribe accurately.

---

## Pharmacist Stories

**US-03**  
As a pharmacist, I want to verify prescriptions digitally so that I can improve medication accuracy.

**US-04**  
As a pharmacist, I want stock validation alerts so that I can avoid shortages.

---

## Patient Stories

**US-05**  
As a patient, I want access to prescription history so that I can track treatments.

**US-06**  
As a patient, I want refill reminders so that I do not miss medication schedules.

---

## Admin Stories

**US-07**  
As an admin, I want system reports so that I can monitor operational performance.
